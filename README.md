# Go News App

Small web app to get news about certain topic you want to search for

## Requirements

The only requirement is to set up an API key at [news.org](https://newsapi.org/) then paste it in `.env`
as `API_KEY=<YOUR_KEY>`

## Run locally

```
go build && ./go-news-app
```
